# which one
List the scripts in package.json and choose one to execute

## Install

```shell
npm install -D @oasis-cloud/which-one
```

## Example
```shell
└─> wo 
? which one › - Use arrow-keys. Return to submit.
❯   test
    color
```
 or
```shell
└─> wo color
? which one › - Use arrow-keys. Return to submit.
❯   color
```
