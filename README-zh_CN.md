# which one
列出 package.json 中的 scripts，选择其中某一项运行。

## Install

```shell
npm install -D @oasis-cloud/which-one
```

## Example
```shell
└─> wo 
? which one › - Use arrow-keys. Return to submit.
❯   test
    color
```
 or
```shell
└─> wo color
? which one › - Use arrow-keys. Return to submit.
❯   color
```
